Pegasus City Guide API

Introduction

The Pegasus City Guide API is a Node.js and Express application that provides users with access to information about different activities available in the city of Dallas. Users will be able to create an account, update their personal information, add and remove items from a to-do list, and see their local favorites.

Endpoints

Request URL Method
Login /login POST
Allow new users to register /users POST
Allow users to view their profile /users/:id GET
Allow users to update their user info /users/:id PUT
Allow existing users to delete their account /users/:id DELETE
Return a list of all activities to the user /activities GET
Add activity to to-do list /users/:id/activities/:activitiesId POST
Remove activity from to-do list /users/:id/activities/:activitiesId DELETE
Display all completed activities /users/:id/activities/:completed GET
Add activity to completed list /users/:id/activities/:completed POST
Remove activity from completed list /users/:id/activities/:completed DELETE

Endpoints in detail

Login

Using https://city-guide-api-3d2f74a4c59e.herokuapp.com/login will authenticate the user and return a JSON object listing all the user data and their issued JWT to access the rest of the site. The query requires a username and password as parameters. For example, https://city-guide-api-3d2f74a4c59e.herokuapp.com/login?Username=example&Password=example

Allow new users to register

Using https://city-guide-api-3d2f74a4c59e.herokuapp.com/users with the POST method will return a JSON object showing the new user's information (as well as an automated ID). The request body must be a JSON object in the following format: { "Username": String, "Password": String, "Email": String }. The query requires no parameter. Entering a valid JSON object in the request body will return the JSON response object. An example of a valid entry is: { "Username": "example", "Password": "example", "Email": "example@email.com" }.
