Pegasus City Guide API

Introduction

The Pegasus City Guide API is a Node.js and Express application that provides users with access to information about different activities available in the city of Dallas. Users will be able to create an account, update their personal information, add and remove items from a to-do list, and see their local favorites.

Endpoints

Request URL Method
Login /login POST
Allow new users to register /users POST
Allow users to view their profile /users/:id GET
Allow users to update their user info /users/:id PUT
Allow existing users to delete their account /users/:id DELETE
Return a list of all activities to the user /activities GET
Add activity to to-do list /users/:id/activities/:activitiesId POST
Remove activity from to-do list /users/:id/activities/:activitiesId DELETE
Display all completed activities /users/:id/activities/:completed GET
Add activity to completed list /users/:id/activities/:completed POST
Remove activity from completed list /users/:id/activities/:completed DELETE

Endpoints in detail

Login

Using https://city-guide-api-3d2f74a4c59e.herokuapp.com/login will authenticate the user and return a JSON object listing all the user data and their issued JWT to access the rest of the site. The query requires a username and password as parameters. For example, https://city-guide-api-3d2f74a4c59e.herokuapp.com/login?Username=example&Password=example

Allow new users to register

Using https://city-guide-api-3d2f74a4c59e.herokuapp.com/users with the POST method will return a JSON object showing the new user's information (as well as an automated ID). The request body must be a JSON object in the following format: { "Username": String, "Password": String, "Email": String }. The query requires no parameter. Entering a valid JSON object in the request body will return the JSON response object. An example of a valid entry is: { "Username": "example", "Password": "example", "Email": "example@email.com" }.

Allow users to update their user info

Using https://city-guide-api-3d2f74a4c59e.herokuapp.com/users/:id/ with the PUT method will return a JSON object displaying all of the user's updated information. A user ID is required as the parameter, and the request must include a request body in the following format: { Username: String, (required) Password: String, (required) Email: String, (required) }. For example, https://city-guide-api-3d2f74a4c59e.herokuapp.com/users/64c2a4ec799ddbb40e98a73f/ would have a request body as follows: { "Username": "example1", "Password": "example", "Email": "example@email.com" }. A successful response returns all the existing user data, which will include the updated sections.

Allow existing users to delete their account

Using https://city-guide-api-3d2f74a4c59e.herokuapp.com/users/:id with the DELETE method will return the HTML response: "Your account was deleted." No response body is required. The query requires the user ID as a parameter. The example https://city-guide-api-3d2f74a4c59e.herokuapp.com/users/64c2a4ec799ddbb40e98a73f/ would return the HTML response, "Your account was deleted."

Return a list of all activities to the user

Using https://city-guide-api-3d2f74a4c59e.herokuapp.com/activities will return an array of activity objects in JSON format. The method type is GET and has no response body. The query requires no parameters.

Add activity to to-do list

Using https://movies-app1-3d6bd65a6f09.herokuapp.com/users/:id/activities/:activitiesId with the POST method will return a JSON object with all of that user's data, including their expanded to-do list. The query has no body but must include an existing user ID and an activity ID as parameters. The example https://city-guide-api-3d2f74a4c59e.herokuapp.com/users/64c2a4ec799ddbb40e98a73f/activities/64c153126181b19b214b4314 would return a JSON object displaying all of the user's data, along with their new to-do activity.

Remove activity from to-do list

Using https://movies-app1-3d6bd65a6f09.herokuapp.com/users/:id/activities/:activitiesId with the DELETE method will return a JSON object with all of that user's data, including their shortened to-do list. The query has no body but must include an existing user ID and an activity ID as parameters. The example https://city-guide-api-3d2f74a4c59e.herokuapp.com/users/64c2a4ec799ddbb40e98a73f/activities/64c153126181b19b214b4314 would return a JSON object displaying all of the user's data, along with their shorter to-do list.

Display all completed activities

Using https://movies-app1-3d6bd65a6f09.herokuapp.com/users/:id/completed/ with the GET method will return a JSON object with all of that user's current data. The query has no body but must include an existing user ID as a parameter. The example https://city-guide-api-3d2f74a4c59e.herokuapp.com/users/64c2a4ec799ddbb40e98a73f/completed/ would return a JSON object displaying all of the user's data.

Add activity to completed list
